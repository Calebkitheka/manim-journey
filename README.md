## 🚀 About This Project

This repository documents my **30-day journey** learning [Manim](https://www.manim.community/) (Mathematical Animation Engine), the same tool used by [3Blue1Brown](https://www.3blue1brown.com/) to create beautiful math visualizations.

**Goal:** Create 30 unique animations in 30 days, learning Python, mathematics, physics, and content creation along the way.

**Current Progress:** **22/30 Days Complete!** 🎉
